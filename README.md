# Cognitive Bias Detector — Setup Guide

## Project Structure

```
cognitive-bias-detector/
├── generate_dataset.py   ← Step 1: generate training data
├── train.py              ← Step 2: fine-tune DistilBERT
├── app.py                ← Step 3: Streamlit app
├── requirements.txt
├── biases_dataset.csv    ← generated by Step 1
├── bias_model/           ← generated by Step 2
│   ├── config.json
│   ├── model.safetensors
│   ├── tokenizer files...
│   └── label_map.json
└── .streamlit/
    └── secrets.toml      ← your API key (never commit this)
```

---

## Step 1 — Generate Dataset

1. Get a free Gemini API key: https://aistudio.google.com/app/apikey
2. Set your key:
   ```bash
   export GEMINI_API_KEY="your_key_here"
   ```
3. Run:
   ```bash
   pip install -r requirements.txt
   python generate_dataset.py
   ```
4. You'll get `biases_dataset.csv` with ~900 rows. Takes ~5-10 minutes.

---

## Step 2 — Fine-tune DistilBERT

Make sure your RTX 3060 drivers and CUDA are set up, then:

```bash
python train.py
```

- Training takes ~20-40 minutes on RTX 3060
- You'll see per-epoch accuracy in the terminal
- Aim for >75% validation accuracy
- Model saves to `./bias_model/`

---

## Step 3 — Run Locally

Create `.streamlit/secrets.toml`:

```toml
GEMINI_API_KEY = "your_key_here"
```

Then:

```bash
streamlit run app.py
```

---

## Step 4 — Deploy to Streamlit Community Cloud

1. Push everything EXCEPT `bias_model/` to a public GitHub repo
   - The model folder is too large for GitHub (use Git LFS or HuggingFace Hub — see note below)
2. Go to https://share.streamlit.io → sign in with GitHub
3. Click **New app** → select your repo → set `app.py` as the main file
4. In **Advanced Settings → Secrets**, add:
   ```
   GEMINI_API_KEY = "your_key_here"
   ```
5. Hit **Deploy**

### Hosting the model (important)

The `bias_model/` folder (~250MB) is too big for a standard GitHub push.
Best free option: upload it to HuggingFace Hub (free, public):

```bash
pip install huggingface_hub
huggingface-cli login
huggingface-cli upload YOUR_USERNAME/cognitive-bias-detector ./bias_model
```

Then in `app.py`, change the model load line to:

```python
MODEL_DIR = "YOUR_USERNAME/cognitive-bias-detector"
# transformers will auto-download from HuggingFace Hub
```

This way Streamlit Cloud downloads the model on first boot. Clean, free, professional.
